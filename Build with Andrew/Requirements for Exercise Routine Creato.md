Requirements for Exercise Routine Creator Web App

The web application should fulfill the following requirements:

Input Fields
Provide 3 to 5 input fields with descriptive placeholder text, allowing the user to specify details such as (examples):
Fitness goal (e.g., "Build muscle", "Lose weight", "Increase flexibility")
Available equipment (e.g., "None", "Dumbbells", "Resistance bands")
Preferred workout duration (e.g., "30 minutes", "1 hour")
Days per week (e.g., "3", "5")
Experience level (e.g., "Beginner", "Intermediate", "Advanced")
Generate Button

Include a clearly labeled button that the user clicks to generate an exercise routine based on the entered information.
Output Display Area

Provide an area on the page where the resulting exercise routine is displayed after generation.
General UI/UX

The app should have a clean, simple, and easy-to-use interface.
Placeholders in the input fields should guide users on what information to enter.
Self-Containment

The app must work standalone, entirely in the browser, with no reliance on external resources or network connections.
Responsiveness

The layout should function and display correctly on both desktop and mobile devices.
